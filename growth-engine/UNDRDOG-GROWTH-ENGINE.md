## STAGE 9: MONTHLY LIMITED DROPS (Profit Engine)

### The Drop Model
```
┌─────────────────────────────────────────────────────────────┐
│                   MONTHLY DROP CALENDAR                      │
├─────────────────────────────────────────────────────────────┤
│  MONTH    │  DROP NAME          │  UNITS    │  STATUS       │
│  ─────────┼─────────────────────┼───────────┼─────────────  │
│  Feb      │  000 Clean Slate    │  25,000   │  ✅ LIVE      │
│  Mar      │  Polos              │  TBD      │  🟡 Planning  │
│  Apr      │  Boxer Briefs       │  TBD      │  🟡 Planning  │
│  May      │  Dark Blue          │  TBD      │  📋 Planned   │
│  Jun      │  Gray               │  TBD      │  📋 Planned   │
└─────────────────────────────────────────────────────────────┘
```

### Drop Sequence
```
WEEK -2: Teaser (email + SMS + social)
  └─► "Something's coming..."

WEEK -1: Reveal (email + SMS + social)  
  └─► Color/product reveal, date announcement

DAY -1: VIP Early Access (email + SMS)
  └─► 24-hour head start for email list

DROP DAY: Public Launch
  └─► Full marketing push
  └─► Scarcity messaging ("X units remaining")
  └─► Real-time inventory updates

WEEK +1: Last Chance
  └─► "Almost gone" email sequence
  └─► FOMO push

POST-DROP: Sold Out
  └─► Waitlist for restock (lead capture)
  └─► "Miss out? Get early access next time"
```

### Why Drops Work
- **Scarcity:** Limited units create urgency
- **Collectibility:** Each drop is unique
- **Retention:** Recurring purchase behavior
- **Profitability:** No acquisition cost (existing customers)
- **Community:** "Did you get it?" social proof
